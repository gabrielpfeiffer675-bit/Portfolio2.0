Gestion de Stock Web

2025 Développement Web Projet Solo

Une application complète pour gérer les flux logistiques d'une PME fictive. Développée en PHP natif avec une architecture MVC.

Contexte & Objectifs

L'objectif était de créer une interface web permettant aux magasiniers de gérer les entrées et sorties de produits. Le système devait être sécurisé et permettre une gestion des droits utilisateurs (Admin vs Magasinier).

Extrait de Code (Connexion PDO)

Voici comment j'ai sécurisé la connexion à la base de données en utilisant le pattern Singleton :

class Database { private static $instance = null; public static function getConnection() { if (self::$instance === null) { try { self::$instance = new PDO("mysql:host=localhost;dbname=stock", "root", ""); self::$instance->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION); } catch(PDOException $e) { die("Erreur de connexion : " . $e->getMessage()); } } return self::$instance; } }
